Aggiungi una piattaforma cloud al tuo sito!

== Description ==

Il plugin offre la possibilità agli utenti del sito, di mantenere sempre a portata di mano i file più importanti, caricarli, scaricarli, gestirli o eliminarli in ogni momento, in ogni luogo e da qualsiasi piattaforma. Un servizio semplice ma efficace per non perdere i documenti personali. Dotato di un'interfaccia intuitiva con cui è possibile accedere ai propri file in modo semplice e rapido.

È inoltre possibile rendere pubblici i file in modo che tutti i visitatori del sito possano scaricarli e pertanto creare un gruppo di lavoro tra più utenti. È anche possibile mantenere la riservatezza dei propri documenti caricati in piattaforma rendendoli invisibili agli altri utenti.
Per quanto concerne gli aspetti di sicurezza e protezione della piattaforma il plugin prevede:

- Prevenzione attacchi SQL injection
- Crittografizzazione MD5 di tutti i dati immagazzinati nel database
- Controlli incrociati sui permessi degli utenti

== Installation ==

1. Scarica il plugin
2. Carica ed installa dal pannello amministratore 'Plugin' di Wordpress
3. Attiva il plugin
4. Inizia a caricare i files dalla pagina 'Inserisci Documento' del tuo sito
5. Guarda ed approva tutti i files dall'apposita pagina 'Impostazioni Cloud' dalla pagina amministratore di Wordpress
